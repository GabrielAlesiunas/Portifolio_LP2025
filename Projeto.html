Projetos Pessoais

Aqui estão alguns dos meus projetos:

Projeto 1: Sistema de Gerenciamento automático de plantas

Desenvolvi um sistema web para ajudar as pessoas a cuidar de suas plantas. O sistema permite controlar a temperatura, umidade da planta. Com isso o sistema faz o gerenciamento automático da planta.

  • Funcionalidades:
  • Monitorar a planta
  • Ver históricos da planta

Projeto 2: Sistema para controle de Produtos Orgânicos

Um projeto desenvolvido em java que faz o gerenciamento de estoque de produtos orgânicos, controla o caixa e o fornecedor dos produtos.

  • Funcionalidades:
  • Controlar o estoque dos produtos
  • Controlar o caixa
  • Acompanhar os fornecedores
